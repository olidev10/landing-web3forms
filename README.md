# Simple Landing Page (HTML/CSS + Web3Forms)

A minimal landing page built with **pure HTML and CSS** that demonstrates two common website features:

- A **button redirecting to another website**
- A **working contact form** that sends emails using **Web3Forms**

The project is designed to be easily deployed on **Vercel or GitHub Pages** without any backend or JavaScript.

---

## Features

- Pure **HTML & CSS**
- **No JavaScript**
- Contact form powered by **Web3Forms**
- Simple **redirect button**
- Ready for **static deployment**
- Clean and minimal UI
