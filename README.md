# Nova Landing (Pure HTML/CSS + Web3Forms)

A simple, modern landing page you can deploy on GitHub Pages or Vercel.
Includes a working contact form using Web3Forms (no JS, no backend).

## Setup
1. Open `index.html`
2. Replace:
   - `PASTE_YOUR_ACCESS_KEY_HERE` with your Web3Forms key
   - `https://YOUR_DOMAIN/thanks.html` with your real deployed URL

Example:
`https://my-landing.vercel.app/thanks.html`

## Deploy to Vercel
- Push this folder to GitHub
- In Vercel: New Project → Import the repo → Deploy

## Notes
- The `botcheck` field is a honeypot (anti-spam).
- You can change colors in `styles.css` under `:root`.
